Добавление и редактирование рецептов

Добавление

Для того, чтобы добавить новый рецепт нажмите кнопку "Добавить рецепт". Откроется редактор рецептов


Здесь вы можете задать название своего рецепта, его описание. Для того, чтобы добавить фото рецепта перетащите фото в место, где нарисована камера. Для того, чтобы добавить ингредиент в список, нажмите кнопку "Добавить ингредиент". Когда закончите описывать рецепт нажмите "Сохранить и перейти к приготовлению". Откроется редактор приготовления.


Здесь вы можете описать все шаги, необходимые для приготовления блюда, а также дополнить их фотографиями результата. Когда закончите, нажмите кнопку "Сохранить изменения".

Редактирование

Чтобы отредактировать рецепт, выберите его в списке и нажмите кнопку "Редактировать рецепт". Откроется редактор рецептов. Дальнейшие действия схожи с добавлением рецепта.
